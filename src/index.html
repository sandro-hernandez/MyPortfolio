TWITCHCOMM: A Community-Based Recommender System for Twitch Users

Technologies: React, Django, Electron, Network Analysis, Link Prediction, Comunities detection Algorithms.
TWITCHCOMM is a recommender system designed for the Twitch platform to help users find new creators to follow based on their social connections. I led the development of the system’s core recommendation algorithms, which utilized Python for community detection and link prediction. Specifically, we employed the Leiden algorithm to detect communities within Twitch’s user network and integrated Node2Vec embeddings for link prediction. For users outside our dataset, we used a popularity-based recommendation approach. The system was built using Python for algorithm development, React and Django for the application, and deployed using Electron. This project culminated in a research paper presented at the SDS2024 conference, showcasing its effectiveness in providing tailored recommendations on a large-scale platform like Twitch.

GitHub Actions Developer Needs

Technologies: Python, SQL, PostgreSQL, Empirical Data Analysis
This ongoing project analyzes GitHub Actions (GHA) related posts on Stack Overflow to identify developer information needs when working with CI/CD pipelines. I conducted an empirical analysis, manually categorizing thousands of posts to pinpoint key themes such as troubleshooting, configuration issues, and workflow optimization. The project uses Python for data processing and analysis, and the results are stored in a PostgreSQL database. We aim to publish this research in a peer-reviewed conference to provide insights for tool creators and the broader development community to improve documentation and support for GHA users.

IBM Final Capstone: Predicting SpaceX Rocket Landing Success Using Machine Learning

Technologies: SQL, REST API, Web Scraping, Pandas, Scikit-learn, Plotly, Dash, Folium, Python.
In this project, I analyzed SpaceX’s Falcon 9 rocket data to predict the success of the first stage landing. The project involved a comprehensive workflow, including data collection from a REST API and web scraping, followed by data wrangling, exploratory data analysis, and building machine learning models. I employed classification techniques such as Logistic Regression, SVM, Decision Trees, and K-Nearest Neighbors. The final models achieved an accuracy of up to 83.3%, with Logistic Regression, SVM, and KNN performing the best. The project also involved building an interactive dashboard using Plotly Dash to visualize launch outcomes and payload data, as well as creating an interactive map using Folium to display launch sites.
